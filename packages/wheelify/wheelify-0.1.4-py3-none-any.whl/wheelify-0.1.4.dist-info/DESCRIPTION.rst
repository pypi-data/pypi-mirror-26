Wheelify
========

Simple utility to build manylinux wheels from a requirements file.

Usage
-----

.. code-block:: shell

    wheelify - Manylinux builder

    positional arguments:
      requirements_file

    optional arguments:
      -h, --help            show this help message and exit
      --user USER
      --wheel-dir WHEEL_DIR
      --python {python2.7,python3.6}
