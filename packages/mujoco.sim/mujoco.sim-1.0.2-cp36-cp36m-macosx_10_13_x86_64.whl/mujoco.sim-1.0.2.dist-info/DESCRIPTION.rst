MuJoCo for Python!
==================

This is a simple python wrapper around the MuJoCo physics simulation. I wrote it when I couldn't get `mujoco-py 
<https://github.com/openai/mujoco-py>`_ to work. 

Installation
------------

.. code-block:: bash

  pip install mujoco
